# 饿了么前端

`b` 站白嫖的教程, 学习了一下, 受益匪浅, 但是只是实现了一下前端布局, 有很多东西都没有实现

1. 没有使用老师写的那个滚动插件, 而是自己实现了一个比较简单的滚动插件(可能有兼容性问题)
2. 尝鲜了 `vux` ui
3. 没有使用的 `vuex`

```console
git clone https://github.com/bilibili-live/ele.git
cd ele
yarn install
```
